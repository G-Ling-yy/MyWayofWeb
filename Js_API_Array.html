Array.isArray()：

Array.isArray(obj) 判断obj对象是否为数组，返回布尔值。

Array.prototype.forEach()

arr.forEach(function(item,index,arr){},[thisArg]) 遍历arr数组操作，thisArg为可选参，以设置callback中的this值，不修改原数组。

Array.prototype.push()

arr.push(elements) 把一个或多个元素添加到arr数组的尾部，如果添加元素有数组则把整个数组作为一个元素添加。

Array.prototype.pop()

arr.pop() 删除并返回arr数组的最后一个元素。

Array.prototype.unshift()

arr.unshift(elements) 把元素添加到arr数组的头部，使用形同push()。

Array.prototype.shift()

arr.shift() 删除并返回arr数组的第一个元素。

Array.prototype.splice()

arr.splice(index,length,[item1]···[itemn]) 修改原数组，删除index处length个元素，同时在该位置添加item系列元素如果有的话，item系列为可选参。

Array.prototype.concat()

arr.concat(elements) 拼接数组，把一个或多个元素或数组依次拼接到arr数组后面，返回一个新数组，不修改原数组。

Array.prototype.reverse()

arr.reverse() 翻转数组，根据index下标颠倒数组，返回新数组，会改变原数组。

Array.prototype.join()

arr.join(string) 把数组中所有元素根据string参数值连接到一个字符串中，默认 ',' 隔开，string空字符串 '' 连接无间隔，不修改原数组。

Array.prototype.some()

arr.some(function(item,index,arr){},[thisArg]) 判断arr数组是否有部分符合条件，只要有ture则true，返回布尔值，thisArg是可选参为指定functino中的this，不修改原数组。

Array.prototype.every()

arr.every(function(item,index,arr){},[thisArg]) some()的全面升级版，全ture为true，否则false，返回布尔值，不修改原数组。

Array.prototype.sort()

arr.sort([function(a,b){return a-b}]) 数组排序函数，函参不写默认Unicode排序，函参写了按返回值大于小于0渐大或渐小排序，会改变原数组。

Array.prototype.slice()

arr.slice([start,end]) 浅拷贝arr数组的指定部分，无参拷贝整个数组，无end拷贝到数组最后，有参拷贝start到end部分不包括end，返回新数组，不会修改原数组。

Array.prototyoe.indexOf()

arr.indexOf(searchElement,[startIndex=0]) 查询指定元素在arr数组的首次出现下标，searchElement不能直接写为数组，只能以插入数组的变量代替，查找起始位置startIndex是可选参默认为0，若超过arr.length - 1不查询。

Array.prototype.lastIndexOf()

arr.lastIndexOf(searchElement,[startIndex=0]) 查询指定元素在arr数组的末次出现下标，用法形同indexOf()。

Array.prototype.find()

arr.find(function(item,index,arr){},[thisArg]) 找出arr数组中首个符合条件的元素并输出，用法形同some()和every()，不修改原数组。

Array.prototype.findIndex()

arr.findIndex(function(item,index,arr){},[thisArg]) 找出arr数组中首个符合条件元素的下标并输出，用法形同楼上find()。

Array.prototype.map()

arr.map(function(item,index,arr){},[thisArg]) 返回arr数组每个元素经过函数function之后返回值组成的新数组，俗称映射~，返回新数组，不修改原数组。

Array.prototype.filter()

arr.filter(function(item,index,arr){},[thisArg]) find()的升级版，输出arr数组中所有符合条件元素，返回新数组，不修改原数组。

Array.prototype.reduce()

arr.reduce(function(accumulator,item,index,arr){},[initialValue]) 从左到右把arr数组中每个元素通过函数function将arr数组缩减成一个单体值，accumulator参为初始值，亦是上次调用返回的累积值，initialValue为可选参是accumulator的第一次的值，返回累积缩减的结果，不修改原数组。

Array.prototype.reduceRight()

arr.reduceRight(function(accumulator,item,index,arr){},[initialValue]) 作用及用法形同楼上reduce()，只是方向相反，为从右到左。

Array.prototype.fill()

arr.fill(value,[start,end]) 填充函数，把value参值根据start和end下标(不包括end)复制填充arr数组，start和end为可选参，默认为0和arr.length，返回修改后数组，修改原数组。

Array.prototype.copyWithin()

arr.copyWithin(target,[start,end]) 内部替换函数，把target参下标元素用根据start和end下标选中的部分元素替换掉，satrt和end用法形同楼上fill()，不修改数组长度，返回修改后数组，修改原数组。

Array.prototype.includes()

arr.includes(value,[startIndex]) 判断arr数组中是否有value参值元素，startIndex为可选参默认为0，返回布尔值，不修改原数组。

Array.prototype.entries()

arr.entries() 返回一个新的Array Iterator对象，该对象包含数组中每个索引的键/值对。

Array.prototype.keys()

arr.keys() 返回一个新的Array迭代器，它包含数组中每个索引的键。

Array.prototyope.values()

arr.values() 返回一个新的 Array Iterator 对象，该对象包含数组每个索引的值(chrome还不支持···尼玛)。

Array.from()

arr.from(arrayLike,[mapFn([thisArg])]) 从一个类似数组或可迭代的对象中创建一个新的数组实例。

Array.of()

Array.of(item1[···itemn]) 创建一个不定参数量的新数组实例，不考虑参数的数量或类型，和Array()构造函数区别就是参数意义不同，Array.of()参数就是value值，Array()参数是数组长度。

Array.prototype.toString()

arr.toString() 效果等同于arr.join()。

Array.prototype.toLocaleString()

arr.toLocaleString() arr数组元素各自调用各自的toLocalString()再拼接成完整字符串。
