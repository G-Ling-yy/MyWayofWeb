String.prototype.trim()

str.trim() 字符串首尾去空格处理，返回新字符串，不修改原字符串。

String.prototype.trimLeft()

str.trimLeft() 字符串头部去空格处理，用法形同楼上trim()。

String.prototype.charAt()

str.charAt(index) 返回指定下标index处字符值，index超值返回空字符串 '' 。

String.prototype.charCodeAt()

str.charCodeAt(index) 返回指定下标index处字符的Unicode编码（不就是ascii码表么）

String.prototype.endsWith()

str.endWith(value,[end]) 判断str的end(不包括)下标处之前是否以value结尾,返回布尔值。

String.prototype.startsWith()

str.startWith(value,[start]) 用法形同楼上endWith()，作用是判断开头,start默认0。

String.prototype.includes()

str.includes(value,[start]) 判断str是否包含有value在start指定位置往后，区分大小写，返回布尔值，start默认0

String.prototype.indexOf()

str.indexOf(value,[start]) 返回value在str中首次出现下标在start指定位置往后，没有则返回 -1 ，start默认0

String.prototype.lastIndexOf()

str.lastIndexOf(value,[end]) 用法形同楼上indexOf()，从后向前找，没有则返回 -1 ，end默认arr.length，负值视为0，超值视为length。

String.prototype.link()

str.link(url) 生成<a>链接，url是href值。

String.prototype.match()

str.match(string|regexp) 根据字符串string参值或正则regexp参值匹配str首个符合要求字符串并返回对应值数组，正则regexp参里若有g参则返回所有匹配值数组，无参返回空数组，无匹配返回null。

String.prototype.search()

str.search(string|regexp) 作用用法形同楼上match()，只是返回首次索引值，没有则返回 -1 。

String.prototype.repeat()

str.repeat(count) 字符串复写count次，返回新字符串，不修改原字符串。

String.prototype.replace()

arr.replace(string|regexp,newString|function) 替换字符串，根据string参值或regexp参值获取首个符合条件元素，用newString或function返回值替换掉（替换字符串中"$"特殊），regexp中有g参则全部替换，返回新字符串，不修改原字符串。

String.prototype.slice()

str.slice([start,end]) 复制str数组指定部分并返回新字符串，不包括end，默认0到length，不修改原数组。

String.prototype.split()

str.split(separator,[maxlength]) 把str字符串根据separator参值分割成长度不超过maxlength的数组，返回新数组，不修改原数组。

String.prototype.substr()

str.substr([start,length]) 作用用法形同slice()，唯第二个参为长度耳，不改原数组。

String.prototype.substring()

str.substring([start,end]) 作用用法和slice()基本无异，唯参数灵活性更大些。

String.prototype.toLowerCase()

str.toLowerCase() str全部小写，返回新字符串，不修改原数组。

String.prototype.toUpperCase()

str.toUpperCase() str全大写，形上。
